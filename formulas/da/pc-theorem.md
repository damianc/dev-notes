# Pseudocode Theorem

## Conditions

| Symbol | Meaning |
|--|--|
| $p \Rightarrow q$ | if `p` then `q` |
| $p \nRightarrow q$ | if not `p` then `q` |
| $p \Leftarrow q$ | `p` if `q` |
| $p \nLeftarrow q$ | `p` if not `q` |
| $p \Leftrightarrow q$ | `p` if `q` |
| $p \nLeftrightarrow q$ | `p` if not `q` |
