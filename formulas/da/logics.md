# Logics

| | | | | | |
|--|--|--|--|--|--|
|| $p$ | 0 | 0 | 1 | 1 |
|| $q$ | 0 | 1 | 0 | 1 |
| AND | $p \land q$ | 0 | 0 | 0 | 1 |
| OR | $p \lor q$ | 0 | 1 | 1 | 1 |
| XOR | $p \otimes q$ | 0 | 1 | 1 | 0 |
| NAND | $p \cancel{\land} q$ | 1 | 1 | 1 | 0 |
| NOR | $p \cancel{\lor} q$ | 1 | 0 | 0 | 0 |
| XNOR | $p \cancel{\otimes} q$ | 1 | 0 | 0 | 1 |
| NOT | $\lnot p$ | 1 | 1 | 0 | 0 |
|| $\lnot q$ | 1 | 0 | 1 | 0 |
